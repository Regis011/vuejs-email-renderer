# Email Renderer

Create email friendly html using Vue.js, Vue Server Renderer & mjml.

## Requirements

- Node v12.18+
- npm v6.14.6+
- yarn 1.22+

## Project setup

``` bash
yarn install
```

## Create email using mock data

``` bash
# ouputs to dist/buildHtmlWithMockData.js
yarn dev
```

## Run unit test

``` bash
yarn test:unit
```

## Test the performance of the render

``` bash
node ./test/performance/renderer.js
```
